Exercice 7 - Local Storage

Utiliser le localstorage afin de stocker des paramètres de configuration.

Grâce aux input, il est possible d'entrer une valeur pour le nom et pour la largeur de la division.

  • Le fichier CSS et HTML ne doivent pas être modifiés.
  • La largeur ainsi que le nom doivent être sauvés dans la mémoire.
  • Dans le cas où une valeur existe en mémoire, il faut l'utiliser lors du chargement de la page.
