Exercice 5

A chaque clique sur le bouton, ajouter ou retirer la classe c3 de la division.
Le texte du bouton doit changer pour indiquer l'action à l'utilisateur.
La division supérieure doit afficher la liste des classes présente dans la division div1

  • Le fichier CSS et HTML ne doivent pas être modifié.

Aides

  • Dans le débogueur du navigateur, aller dans l'onglet "Debogueur".
  • Sélectionner le fichier ex5.js et placer un point d'arrêt pour analyser le code.
  • Comme vous pouvez le voir dans la fonction, une variable window.test est utilisée.
    Essayer d'entrer window.test dans la console du débogueur.
  • Chercher les propriétés et les méthodes qui pourraient être utile dans l'objet "window.test" pour manipuler les classes de la division.
